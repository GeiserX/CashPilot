## Description

Sentinel is a Layer 1 Cosmos blockchain for decentralized VPN. Node operators sell bandwidth to VPN users with flexible pricing (per-GB or hourly). Official Docker deployment guide available. Long-running project with active community.

## Setup Instructions

### 1. Create an account

Sign up at [Sentinel dVPN](https://sentinel.co).

### 2. Get your credentials

After signing up, locate the credentials needed for Docker deployment. These are typically your email/password or an API token found in the dashboard.

### 3. Deploy with CashPilot

In the CashPilot web UI, find **Sentinel dVPN** in the service catalog and click **Deploy**. Enter the required credentials and CashPilot will handle the rest.
